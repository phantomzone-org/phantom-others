1. Download and install sage
2. git clone `lattice-estimator` inside `noise` directory:

```
cd noise
git clone https://github.com/malb/lattice-estimator.git
```

3. Move all files in `lattice-estimator` folder to parent directory `noise`:

```
cd lattice-estimator
move * ../
```

4. Run the script `sage tester.py`
